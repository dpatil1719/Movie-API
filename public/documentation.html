Movie API Documentation

Method Endpoint Description Request Body (JSON)
GET /movies Return a list of ALL movies None
GET /movies/:title Return data about a single movie by title (description, genre, director, image URL, featured) None
GET /genres/:name Return data about a genre by name (description) None
GET /directors/:name Return data about a director (bio, birth year, death year) by name None
POST /users Allow new users to register 
{
  "Username": "alice",
  "Password": "password123",
  "Email": "alice@example.com",
  "Birthday": "1995-01-01"
}
PUT /users/:username Allow users to update their user info 
{
  "Username": "alice",
  "Password": "newpass456",
  "Email": "alice_new@example.com",
  "Birthday": "1995-05-05"
}
POST /users/:username/movies/:movieId Allow users to add a movie to their list of favorites None
DELETE /users/:username/movies/:movieId Allow users to remove a movie from their list of favorites None
DELETE /users/:username Allow existing users to deregister None
