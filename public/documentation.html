Movie API

Welcome to the Movie API! This is a RESTful API that provides access to information about movies, directors, and genres. Developers can use this API to retrieve data about popular movies, search for specific titles, explore director biographies, and browse movie genres. The API supports common HTTP methods like GET, POST, PUT, and DELETE to interact with and manage movie data.

Please refer to the sections below for detailed information on API endpoints, authentication, and example usage.

API Endpoints

Request URL Response
GET /movies Returns a list of all movies
GET /movies/:title Returns data about a single movie by title
GET /genres/:name Returns data (description) about a genre by name
GET /directors/:name Returns data (bio, birth/death year) about a director by name
POST /users Allows a new user to register
PUT /users/:username Allows users to update their username
POST /users/:username/movies/:movieId Adds a movie to user's list of favorites
DELETE /users/:username/movies/:movieId Removes a movie from user's list of favorites
DELETE /users/:username Deletes the user account
