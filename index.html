Movie API Documentation

Method Endpoint Description Request Body
GET /movies Return a list of ALL movies None
GET /movies/:title Return data about a single movie by title None
GET /genres/:name Return genre details (description) by name None
GET /directors/:name Return director details (bio, birth year, death year) by name None
POST /users Register a new user 
{
  "Username": "user1",
  "Password": "securepass",
  "Email": "user1@example.com",
  "Birthday": "1990-01-01"
}
PUT /users/:username Update an existing user's info 
{
  "Username": "newName",
  "Password": "newPass",
  "Email": "newEmail@example.com",
  "Birthday": "1985-05-05"
}
POST /users/:username/movies/:movieId Add a movie to a user's favorites None
DELETE /users/:username/movies/:movieId Remove a movie from a user's favorites None
DELETE /users/:username Deregister (delete) a user None
