Movie API Documentation

Request Endpoint Description Request Body
GET /movies Return a list of ALL movies None
GET /movies/:title Return data about a single movie by title (description, genre, director, image URL, featured) None
GET /movies/genre/:genreName Return data about a genre by name (description) None
GET /movies/director/:directorName Return data about a director (bio, birth year, death year) None
POST /users Allow new users to register { "username": "user", "password": "pass", "email": "email" }
PUT /users/:username Allow users to update their user info { "username": "newName", "password": "newPass", "email": "newEmail" }
POST /users/:username/favorites/:movieID Allow users to add a movie to their list of favorites None
DELETE /users/:username/favorites/:movieID Allow users to remove a movie from their list of favorites None
DELETE /users/:username Allow existing users to deregister None
